Hi people of the internet — this website provides information about various sci-fi movies from the 50s to the 80s, including *The Creature from the Black Lagoon*, *Invasion of the Saucer Men*, *Star Wars IV* and many more.

Please be aware that **this site is a hobby catalogue — no movies are sold here**.

---

**About this version:** This is a recreation of a PHP/MySQL project originally built as a TAFE assignment circa 2009. The original site ran on Apache with a MySQL database; this version has been reimagined as a static site using [Hugo](https://gohugo.io) and is hosted on [Netlify](https://netlify.com). The migration from PHP to Hugo was performed with the assistance of [Claude](https://claude.ai) (Anthropic's AI assistant), which analysed the legacy codebase, extracted the movie data from the SQL dumps, and generated the Hugo templates, content files, and data files that power this site.
